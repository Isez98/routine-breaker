# Daily Routine Randomizer

A React-based web application that helps users create spontaneous, randomized daily schedules by selecting random activities from predefined categories.

## Features

- **Category Management**: Create custom categories with multiple location options
- **Smart Randomization**: Randomly selects locations from each category for variety
- **Interactive Maps**: Uses OpenStreetMap + Leaflet for route visualization
- **Export to Maps**: Open routes in Google Maps or Apple Maps
- **Share Routes**: Share your daily routine with others
- **Local Storage**: Saves your categories for future use

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000 in your browser
